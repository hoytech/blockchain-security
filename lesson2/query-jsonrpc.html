When you started ganache-cli, it printed out a list of addresses that were given a default balance.
Query ganache-cli's JSON-RPC port to find the balance of one of these addresses.
An easy way is to use the curl command: 
$ curl http://localhost:8545 -d '{"method":"eth_getBalance","params":["ADDRESS_GOES_HERE"]}'
The output is in wei (there are 1e18 wei in 1 ETH), and is encoded in hexadecimal. Convert this to ETH and decimal.
