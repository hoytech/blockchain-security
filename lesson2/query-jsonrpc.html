Lab 3: Query JSON RPC Node

Write a script in a language of your choice to find the balance in ETH of an account by querying a running ganachi-cli instance's JSON-RPC port (8545). Include the script in your submission.
  • You need to do an HTTP POST request to http://localhost:8545
  • You will use the eth_getBalance RPC method
  • The body of the POST should be JSON, and should look like: 
    {"method":"eth_getBalance","params":["ADDRESS_GOES_HERE"]}
  • The output is in wei (there are 1e18 wei in 1 ETH), and is encoded in hexadecimal. Convert this to ETH and decimal.
  • When you started ganachi-cli it printed out some addresses that were given ETH. Try to query those.
  • What about other addresses? Does it throw an error for unknown addresses?
